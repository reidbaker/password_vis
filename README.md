This uses [PyTagCloud][1] to create a word cloud of the plain
text password leaks from yahoo and billabong.


To run first install [PyTagCloud][1] 
Then clone Repo
>`git clone https://github.com/reidbaker/password_vis.git`

Then run vis.py
>`python vis.py`
